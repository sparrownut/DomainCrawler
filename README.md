# DomainCrawler
内链爬虫 - 找到深度埋藏的资产  
在输入文件中 一行一个想要爬取的目标资产的url  
爬取后会保存为%ParentDomain%.csv的格式   
EG:  
--list.txt  
----https://m.example.com  
----https://www.example.com
